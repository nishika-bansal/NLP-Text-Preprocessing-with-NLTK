# NLP Text Preprocessing Pipeline for Low-Resource Language Processing

This project is a Natural Language Processing (NLP) text preprocessing tool built using Python and NLTK. It cleans and prepares raw text data for NLP tasks by performing tokenization, stopword removal, stemming, and lemmatization. The project is designed with a focus on basic preprocessing techniques and can be extended for low-resource and multilingual language processing such as Hindi-English text.

## Features
- Tokenization using NLTK  
- Stopword removal  
- Duplicate word filtering  
- Stemming and lemmatization  
- Text cleaning and normalization  
- Saves processed output to file  

## Technologies Used
- Python  
- NLTK  

## How to Run
pip install -r requirements.txt  
python preprocess.py  

## Future Scope
Can be extended for Hindi/Hinglish text preprocessing and further NLP tasks like sentiment analysis or text classification.
