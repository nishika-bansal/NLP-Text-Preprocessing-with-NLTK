# NLP Text Preprocessing Pipeline

This project is a Python-based Natural Language Processing (NLP) tool using NLTK. It performs text cleaning and preprocessing to prepare raw data for NLP tasks.

## Features
- Tokenization
- Stopword removal
- Duplicate removal
- Stemming
- Lemmatization
- Word frequency analysis

## Technologies Used
- Python
- NLTK

## How to Run

### 1. Install dependencies
pip install -r requirements.txt

### 2. Run setup (first time only)
python setup.py

### 3. Run project
python preprocess.py

## Input/Output
- input.txt → raw text input
- output.txt → processed output

## Future Scope
Can be extended for Hindi/Hinglish preprocessing and NLP tasks like sentiment analysis.
