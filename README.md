# NLP Text Preprocessing Pipeline

## 📌 Description
This project is a text preprocessing tool built using Python and NLTK.

## 🚀 Features
- Tokenization
- Stopword removal
- Duplicate removal
- Stemming
- Lemmatization
- Save output to file

## 🛠️ Technologies Used
- Python
- NLTK

## ▶️ How to Run
py preprocess.py
